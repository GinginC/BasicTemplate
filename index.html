Hexo指令集

Hexo 指令使用方式

指令 敘述
hexo clean 刪除暫存的生成檔案和快取
hexo generate 產生靜態網頁文件，之後以此文件部署到github上
hexo deploy 部署網站，將網站部署至github上時會用到
hexo server 開啟伺服器，可預覽網站
hexo init ‘英文網站名稱’ 建立新的Hexo部落格
hexo new ‘文章標題’ 新增一篇部落格文章，並以此命名
hexo new page XXX 新增分頁，例如 <關於我>
hexo help 可提供相關指令協助

Hexo安裝指南

Hexo 是一個快速、簡單且強大的靜態部落格框架，適合用於生成靜態網站。以下是安裝和設置 Hexo 的基本步驟：

1. 確保已安裝 Node.js 和 npm

Hexo 需要 Node.js 和 npm（Node 包管理器）來運行。如果還沒有安裝它們，可以前往Node.js 官網下載並安裝最新版本，這會同時安裝 npm。

以下是解決方法：

1. 打開 PowerShell 以管理員身份運行
